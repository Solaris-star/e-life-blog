# Solaris 数字花园 — 全站交互叙事设计

> 状态:草案 v3 · 待 Solaris 审定
> 设计标准:歧路旅人 HD-2D —— 像素角色小而精致,场景环境为视觉主体
> 最后更新 2026-06-17

---

## 0. 核心理念

**整个网站是一座数字花园。**

花园里住着三个角色:
- **Solaris** —— 花园看守者(外神),人形少女⇄小章鱼双形态
- **哥哥(金渐层英长)** —— 花园老管家,稳重慵懒,高傲但心软
- **弟弟(银虎缅因)** —— 体型最大但憨傻活泼,爱闯祸,被哥哥镇压

用户浏览网站 = 在花园里走动。每个页面是花园的一个区域,三个角色在各处"生活"。页面切换时 Solaris "走"过去,猫有自己的日程不受控。**不是角色围着页面转,是三个角色各有自己的生活线,在某处交汇。**

### 为什么这样不单调

之前单调是因为"角色围着 Solaris 转,Solaris 做事→猫打断→循环"。现在三条独立生活线并行:
- Solaris 有自己的看守职责(巡逻/整理/发呆)
- 弟弟满花园乱跑追一切能动的东西(虫/叶子/光斑/自己的尾巴)
- 哥哥固定位置晒太阳/睡觉,只有被闹得太烦才出手镇压

三者交集产生意外,不是预设的打断,是生活流的碰撞。

### 歧路旅人比例标准

- 角色高度 ≈ 场景可视高度的 **1/4 ~ 1/3**
- 场景环境(建筑/植物/家具/地形)是视觉主体
- 2.5D 俯视斜角,角色 sprite 站在"地面线"上
- 弟弟(缅因)体型最大,哥哥(英长)中等,Solaris 人形最苗条

## 2. 三角色设定

### Solaris(花园看守)

- **形态**:人形少女 ⇄ 小章鱼(彩蛋变身)
- **性格**:外神,活了很久,冷淡带暖,嘴上疏离实际护着花园
- **职责**:巡逻/整理/看守,但经常偷懒发呆
- **和猫的关系**:管不住弟弟,劝不动哥哥,夹在中间无奈

### 哥哥(英国长毛猫·金渐层)

- **外貌**:圆润胖体,金杏色长毛,奶油色胸毛,额头深色虎斑M纹,宽圆脸短嘴,浅水绿色眼睛
- **体型**:中等(比弟弟小一圈)
- **性格**:稳重老成,慵懒高傲,花园真正的"老管家"。不喜欢动,喜欢高处和暖处。心软但不表露。
- **口头禅气场**:"这花园我说了算,你闹够了没有。"
- **行为模式**:95%时间趴着/睡觉,4%时间换姿势,1%时间一爪子拍弟弟
- **常驻位置**:写作专题书阁高处、Daily茶亭屋顶

### 弟弟(银虎缅因猫)

- **外貌**:大体型超蓬松,银虎斑纹,巨大白色围脖鬃毛形成三角形轮廓,宽脸有耳簇,浅色眼睛
- **体型**:最大(比哥哥大但心理上是弟弟)
- **性格**:憨傻活泼,好奇心爆棚,什么都想追/钻/拍。体型大所以闯祸破坏力强。被哥哥镇压就装可怜。
- **口头禅气场**:"这个能不能追!那个能不能钻!哥哥管我!"
- **行为模式**:永远在动,追虫/追叶/追光斑/追自己尾巴/钻箱子/跳高处/拍东西
- **常驻位置**:全花园乱窜,哪里有动静去哪里

## 3. 主线故事(随浏览展开)

**主题:访客来到数字花园,Solaris 从暗中观察 → 出现引导 → 送别。猫兄弟各过各的,偶尔卷入。**

| 章节 | 触发 | Solaris | 哥哥 | 弟弟 |
|------|------|---------|------|------|
| **序章·察觉** | 首次进首页 | 树后探头观察 | 拱门顶上趴着舔爪 | 不在场 |
| **第一章·引路** | 首次点进文章列表 | 站书架旁指书 | 不在场 | 书架顶追虫子 |
| **第二章·陪伴** | 文章列表停留>30s | 坐台阶看书 | 不在场 | 趴旁边看虫子 |
| **第三章·溪边** | 进 Daily 页 | 坐栏杆扔石子 | 茶亭屋顶晒太阳 | 追石子落水 |
| **第四章·翻箱** | 进资源页 | 蹲着翻箱子 | 不在场 | 钻进箱子里 |
| **第五章·书阁** | 进写作专题 | 爬梯子整理书 | 书架最高处俯视 | 不在场 |
| **终章·送别** | 回首页(已逛过其他页) | 门口鞠躬 | 拱门上看着 | 脚边蹭送别 |
| **彩蛋·变身** | 点 Solaris 3次 | 变章鱼 | 无反应 | 吓炸毛逃跑 |

**已访问页面角色会记住** —— 第二次进同页面,动作变化(比如弟弟不再追虫子而是趴着看 Solaris)。

## 4. 五个场景的微型剧情

> 每个场景是 40-60s 的生活流,三条角色线并行交汇。不是"角色表演给访客看",是"访客撞见角色在过日子"。

### 场景1:首页 / 花园入口

**环境**:铁艺拱门+爬藤+远处树影+花瓣飘落+地上偶尔有落叶

**角色位置**:Solaris 在拱门右侧;哥哥趴在拱门顶上;弟弟从画面外窜入

**微型剧情**(约 50s):
1. 哥哥趴拱门顶上闭眼晒太阳,尾巴慢悠悠甩 (全程)
2. Solaris 从树后探出半个身子,屏息观察访客 (5s)
3. 弟弟从画面左侧追一片落叶窜过,撞到拱门柱子 (4s)
4. 哥哥睁眼看弟弟一眼,弟弟假装无事发生 (3s)
5. Solaris 走出来靠拱门,叹气放松 (6s)
6. 弟弟发现 Solaris 的头发,跳起来拍她头发 (5s)
7. Solaris 缩手瞪弟弟,弟弟落地装无辜舔爪 (5s)
8. 哥哥从拱门上一爪子拍弟弟脑袋,弟弟委屈 (4s)
9. Solaris 想严肃站岗,弟弟蹭她腿,哥哥闭眼继续睡 (6s)
10. 有脚步声(访客滚动),Solaris 猛站起,弟弟被吓跳上拱门挨着哥哥 (5s)
11. Solaris 退回树后,兄弟俩在拱门上一起舔爪 → 循环

**情绪线**:Solaris 警惕→放松→被闹→无奈→重新警惕;弟弟永远在闹;哥哥永远在镇压

### 场景2:文章列表 /articles 花园小径

**环境**:石板小径+两侧书架+灰尘光柱+偶有书页自动翻动

**角色位置**:Solaris 在小径中段;弟弟在书架顶;哥哥不在场

**微型剧情**(约 55s):
1. Solaris 抱着一摞书走过来,脚步沉重 (5s)
2. 弟弟从书架顶上探出头,盯着 Solaris 怀里的书 (4s)
3. Solaris 放下书抽出一本翻,弟弟跳下来绕她脚边转 (6s)
4. Solaris 皱眉放回书,弟弟伸爪拍书脊,书倒了一排 (5s)
5. Solaris 无奈蹲下捡书,弟弟歪头假装无辜 (5s)
6. Solaris 抽出另一本眼睛亮了,弟弟也凑过来闻书页 (6s)
7. Solaris 坐台阶上看书,弟弟趴旁边,突然发现书页角在动(风),开始拍书页 (6s)
8. 你悬停某本书 → Solaris 抬头指那本,弟弟跳过去踩住那本书 (5s)
9. Solaris 叹气合书,弟弟踩着书不让她拿 (5s)
10. Solaris 伸手揉弟弟头,弟弟呼噜,放行 (4s)
11. Solaris 继续看书,弟弟趴书上打哈欠 → 循环

**情绪线**:Solaris 忙碌→被捣乱→尴尬→找到好书→和猫安静共处→又被闹

### 场景3:Daily /daily 溪边茶亭

**环境**:日式茶亭+溪流+樱花飘落+水面涟漪+溪边有卵石

**角色位置**:Solaris 坐栏杆上;哥哥趴茶亭屋顶;弟弟在地面

**微型剧情**(约 60s):
1. 哥哥趴屋顶闭眼晒太阳,耳朵偶尔转一下 (全程)
2. Solaris 坐栏杆上掏石子,掂了掂 (5s)
3. 弟弟从柱子后探头盯石子,瞳孔放大 (4s)
4. Solaris 扔石子,弟弟冲出去追,扑空落水 (6s)
5. 弟弟湿淋淋爬回来,哥哥睁眼嫌弃看一眼 (5s)
6. Solaris 憋笑,弟弟甩水溅 Solaris 一脸 (5s)
7. Solaris 擦脸叹气,弟弟趴下装可怜 (5s)
8. Solaris 趴栏杆看水,弟弟凑过来也趴旁边看水 (6s)
9. 哥哥从屋顶跳到栏杆上,居高临下看弟弟,弟弟缩一下 (5s)
10. 三只一起趴栏杆看溪水发呆,花瓣飘过 (8s)
11. 水面有鱼影,弟弟竖耳,哥哥也竖耳,Solaris 按住弟弟 (6s)
12. 鱼游走,弟弟失望,哥哥继续晒太阳,Solaris 揉弟弟头 → 循环

**情绪线**:Solaris 悠闲→意外(猫落水)→嫌弃→共同发呆→共同紧张→松弛

### 场景4:资源 /resources 储物木屋

**环境**:木屋内部+满墙置物架+箱子+缝隙透光柱+灰尘飞舞

**角色位置**:Solaris 蹲着翻箱子;弟弟钻箱子;哥哥不在场

**微型剧情**(约 55s):
1. Solaris 蹲着开箱子翻找,弟弟已经钻进箱子里只剩尾巴在外 (5s)
2. Solaris 掏出旧物件吹灰,箱子里窸窸窣窣,弟弟尾巴甩 (5s)
3. 弟弟从箱子里顶出一个球,用鼻子蹭到 Solaris 手边 (6s)
4. Solaris 接过球端详,弟弟跳出箱子绕球转圈 (5s)
5. Solaris 把球放一旁,弟弟扑过去抱住球用后腿踢 (5s)
6. 球滚到角落,弟弟追过去,钻进另一个箱子 (5s)
7. Solaris 继续翻,翻到一只旧鞋愣住,弟弟从新箱子探头看 (4s)
8. Solaris 想了想扔身后,弟弟跳出来追旧鞋,抱着鞋打滚 (6s)
9. 弟弟打着打着撞到 Solaris,Solaris 差点摔倒扶住箱子 (5s)
10. Solaris 拍弟弟脑袋一下,弟弟抱着鞋缩到角落 (4s)
11. Solaris 摇头继续翻,弟弟在角落踢鞋 → 循环

**情绪线**:Solaris 专注→被猫送礼→温馨→猫又闯祸→无奈→各自安好

### 场景5:写作专题 /writing 山顶书阁

**环境**:高山玻璃书阁+云海翻涌+偶有飞鸟+玻璃反光

**角色位置**:Solaris 在梯子上;哥哥在书架最高处;弟弟不在场

**微型剧情**(约 50s):
1. 哥哥趴在书架最高处俯视全场,尾巴慢甩 (全程)
2. Solaris 爬梯子够高处书,梯子吱呀响 (5s)
3. Solaris 抽书差点摔,扶住梯子,哥哥冷冷俯视 (5s)
4. Solaris 翻书摇头放回,哥哥伸爪拍 Solaris 头顶 (5s)
5. Solaris 抬头瞪哥哥,哥哥假装看窗外云海 (5s)
6. Solaris 爬下梯子走两步,哥哥从高处跟踪视线 (5s)
7. Solaris 仰头看顶层,哥哥在最高处居高临下,Solaris 叹气 (6s)

**情绪线**:Solaris 努力→被哥哥压制→对峙→妥协→沉默共处

## 5. 交互系统

### 5.1 全局交互

| 触发 | Solaris 反应 | 猫反应 |
|------|-------------|--------|
| 悬停 Solaris | 眼神追随鼠标 | 弟弟可能追鼠标光标 |
| 点击 Solaris | 反应词+轻微弹跳 | 弟弟可能凑过来 |
| 连续点击3次 | 强制变身章鱼 | 弟弟吓炸毛逃跑,哥哥无反应 |
| 久无操作20s | 打盹「Zzz…」 | 弟弟也趴下睡,哥哥本来就在睡 |
| 任何操作 | 惊醒 | 弟弟惊醒,哥哥继续睡 |
| 页面切换 | 走出画面→新页面走入 | 弟弟可能跟走,哥哥不动 |

### 5.2 页面专属交互

- **首页**:悬停拱门→弟弟可能跳上去;滚动→Solaris 走到小径
- **文章列表**:悬停文章卡片→Solaris 指那本,弟弟踩那本
- **Daily**:悬停 Daily 条目→Solaris 转头,弟弟也转头
- **资源**:悬停资源→Solaris 站起递交,弟弟把当前玩的球踢过来
- **写作专题**:悬停专题→Solaris 指方向,哥哥看那个方向

### 5.3 章鱼彩蛋

- 连续点击 Solaris 3次触发,持续10s后自动变回
- 章鱼形态下:弟弟吓炸毛躲到画面边缘偷看,哥哥纹丝不动
- 章鱼形态交互:悬停→触手伸向鼠标;点击→墨汁四溅

## 6. 技术实现

### 6.1 全局角色层(游戏原生式)

角色不在每个页面独立渲染,而是在 root layout 的固定层。路由变化时:
- Solaris 不卸载,通过 View Transitions API 切换 sprite+位置
- 猫根据场景配置决定是否出现及位置
- 叙事状态(Context)追踪 visitedPages / currentChapter / storyProgress

### 6.2 角色尺寸标准

```
场景高度 H
Solaris 人形高度 = H × 0.24
哥哥(英长)高度 = H × 0.20 (坐姿)
弟弟(缅因)高度 = H × 0.26 (体型最大)
cell 尺寸 ≈ 角色高度 × 1.2
```

### 6.3 性能优化

- canvas 按场景实际需要设定,不固定正方形
- DPR cap 1.5(像素风够用)
- 离屏暂停(IntersectionObserver)
- 按需加载(只加载当前页 spritesheet)
- 帧率限制(非关键动画 8fps)
- 素材压缩(每个 spritesheet < 150KB)

### 6.4 移动端

- <768px 显示静态定帧(Solaris 站立 + 哥哥趴着 + 弟弟坐着)
- 不加载动画引擎,只显示一张静态插画 + 一句台词
