## 2. 思考文字呈现

- 逐字「盖章」显现(复用站点 `stamp-in`)→ 停 2.5s → 墨迹晕散
- 位置:角色**右上角**;白色云朵框(`border-radius: 16px 16px 16px 4px`,右下尖角指向角色)+ 思考引线点 `◦ ∘ °` 从云朵左下朝角色头部递减

## 3. 台词轮换池(每拍随机抽,避免每圈一样)

- ②察觉:「脚步声。新的访客。」/「唔,有人来了。」
- ③打量:「你在找什么?这里什么都有,又什么都没有。」/「随便逛。能读懂多少,看你。」
- ④变身:「想看真正的我?……还太早。」/「别被吓着,我本就不是'人'。」
- ⑤守望:「(漂过来,歪头)」/「小一点,才钻得进字里行间。」
- ⑦收束:「我会在这儿,一直都在。」/「看累了,就歇会儿。」

---

## 4. 互动支线

- **点击她** → 打断当前拍,强制变身 + 反应词(轮换:「……别戳。」/「很痒。」/「还来?」/「好啦好啦。」),重置 30s 计时
- **鼠标靠近** → 章鱼态伸触手探向光标 / 人形态眼神追随;偶发「(盯——)」
- **久不动(~20s 无鼠标移动)** → 打盹,闭眼飘「Zzz」;一动鼠标 → 惊醒,回到 ②察觉
