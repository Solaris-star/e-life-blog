# 欢迎来到我的数字花园

这是一个使用 **Next.js 15**、**Tailwind CSS** 构建的现代博客系统。

## 为什么选择这个架构？

1. **极致的性能**：使用 React Server Components。
2. **完美的心流**：直接在 Obsidian 里写 Markdown，保存即发布。
3. **完全由我掌控**：没有第三方数据库，所有数据都存在本地的 Markdown 文件里。

我们可以像在 Obsidian 里一样直接使用双链：[[关于我]]，这将被渲染为指向 `/articles/关于我` 的链接。
